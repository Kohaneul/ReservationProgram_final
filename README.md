# 방문예약프로그램

### 프로젝트 목적
<ol>전산화를 통한 효율적인 신원확인 (기존 : 장부 수기 작성)</ol>
<ol>외부인 출입 기록 DB 관리 및 유지</ol>

### 대상 : 본사 소속 직원, 경비실

### PROCESS : 접견자가 방문자에 대한 정보를 사전 기입 후 방문시 경비실에서 체크하는 방식

### 개발툴
Java11, SpringBoot, Gradle, MySQL, JavaScript, MyBatis, HTML, CSS, Thymeleaf

### 사용방법  
1. 직원용
<ol>1) 입력</ol>
<ol>2) 조회</ol>
<ol>3) 수정/삭제</ol>
2. 경비실용
<ol>1) 조회</ol>
<ol>2) 방문체크</ol>
